# Nest TypeScript Starter Template

[English](./README.md) / 简体中文

一个固执己见、基于 Nest 和 TypeScript 的启动模板。

## 技术栈

- Nest
- TypeScript
- Prisma
- PostgreSQL

## 使用

### 安装

```bash
pnpm i
```

### 启动

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```
